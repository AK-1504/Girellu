INSTALLATION INSTRUCTIONS:

The data.acd file is intended as a replacement used alongside the KS MX5 ND visuals
or a fitting visual of your choosing. Visuals and sound are not bundled with the package.

How to install:

RECOMMENDED VISUALS AND SOUND

Mazda Miata ND1 Club : KS MX5 ND and MX5 ND sound
Mazda Miata ND1 S1 : KS MX5 ND and MX5 ND sound

Installation instructions

1.
make 2 new folders on content\cars:
arch_mazda_miata_nd1_2016
arch_mazda_miata_nd1_2016_s1

2.
copy from folder ks_mazda_mx5_nd
all files to the new previously created folders:
arch_mazda_miata_nd1_2016
arch_mazda_miata_nd1_2016_s1

3.
Extract and replace all from the downloaded file

4. 
use CM to Replace Sound and select the KS MX5 ND
