## Filter Notes
The name BRUH came from how bad the filter had originally looked.

**BRUH**

Use with Base WFX

**BRUHg**

Use with Base WFX and includes experimental CSP gamma fix

**BRUHgp**

Made for Pure WFX and includes experimental CSP gamma fix
