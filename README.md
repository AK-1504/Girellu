## FAQ
"*What do these folders mean?*"
* "Releases" is where downloadable content reside.
* "Resources" is where general AC information can be read.

"*Why host on GitHub?*"
* Avoid community feuds (Discord servers, RaceDepartment, etc.)
* Easier to host.
* Quicker to host.
